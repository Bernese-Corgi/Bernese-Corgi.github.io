프론트엔드 개발에 필요한 기술

1. 필수적으로 필요한 기술들

html : tag & attributes, semantic web

css : layout ( float , flex, grid ) , transition . animation , 반응형 웹 , Preprocessor (Sass, Less, Stlus, PostCSS) , CSS 방법론, CSS 프레임원크

크로스브라우징

JavaScript : ES5, ES6 (ES6가 ES5를 포함하고 있다.) (바닐라 자바스크립트, 가장 기본이지만 얘만가지고 만들기엔 한계가 있음, 가장 기본이기 때문에 제대로 이해하고 있어야한다.), ES Next, DOM/Event, Ajax, 동작원리 (브라우저 -html. css를 어떻게 해석하는지도 알아야 함-, 자바스크립트 엔진) , node.js

  • let/const 가 변수를 제어하는 함수, 이전에는 var가 있었다. var로 충분하다면 let, const가 나올 이유가 없었을것이다. 즉, var의 문제점을 알아야 한다.
  • ES5도 같은 관점으로 바라보자. 코딩은 ES6로 하고 중점도 그에 두되, ES5의 문제점과 사용법은 알아야한다.

HTTP : 서버와 브라우저 간의 요청/응답하는 동작원리인 http라는 프로토콜에 대해 이해할 필요가 있다.

Tools : Git, Webpack, Babel, ESLint, npm …

툴들을 설치하고 응용할 줄 아는 능력도 중요한 능력이다.

Library / Framework : SPA (Angular, React - 최근 가장 많이 요구되는 프레임워크- , Vue.js), TypeScript, jQuery - 배우진 않을것, 알면좋지 - , Lodash, Axios…

TDD (Testa Driven Development) : karma / jasmine , mocha , chai

  • 대기업일수록 많이 본다.

회사가 신입 개발자에게 요구하는 능력

너무 기초적인 것 (기본적인 용어와 개념, 주력 언어의 기본 문법, 아주 기본적인 알고리즘/자료구조, git 등) 까지 알려주기는 싫다

빨리 배울까? 가르치기 쉬울까? 말을 알아들을까? 하나를 가르치면 열을 알았으면 좋겠다.

회사 분위기를 망치지는 않을까?

말을 잘 들을까?

초심자가 경험하는 3가지의 어려움

  1. 책이나 수업의 내용이 무슨 말인지 하나도 모르겠다. 주변 개발자의 말을 알아 들을 수 없다.

    배경지식 : 기본적 cs(컴퓨터 사이언스) 지식 + 용어에 대한 이해 + 기본 상식

  2. 어떻게 만들어야 할 지 감조차 오지 않는다.

    문제 해결 능력 (problem solving) : 문제(해결과제)가 무엇인지 알아채는 능력 + computational thinking + 알고리즘 / 자료 구조 + 경험

    *문제해결능력 = 해결 과제(문제/요구사항)의 명확한 이해 → 복잡함을 단순하게 분해 → 자료를 정리하고 구분 (modeling) → 순서에 맞게 행위 배열

  3. 어떻게 만들어야 할 지는 알겠는데 막상 코딩하려니까 한 줄도 못짜겠다.

    인터넷에서 긁어온 코드를 도무지 수정할 수 없다

    구현능력 : 문법에 대한 정확한 이해 + 연습

프로그래밍 학습 방법

  1. 하지말아야 할것 : 라이브 코딩 감상, 예제 단순 타이핑, 현재 자신의 능력으로 쉽게 해결 가능한 것을 반복 연습
  2. 자신의 능력을 살짝 넘어서는 도전을 지속적으로 → 쉬운 도전 X
  3. 시행착오 : “ 무엇을 알고 무엇을 모르는지 “
  4. 시도하고 실패하는 의식적인 연습을 반복
  5. 구체적인 목표 , 학습하고 작은 성취를 반복
  6. 본질에서 벗어난 학습 X , 적절한 지점을 찾아내 균형을 잡는 것도 능력이다.
  7. 피드백에 겸손하고 적극적으로 반응

프로그래밍 학습 방향

  1. 기본개념과 동작원리

    이해의 영역 : 용어 / 개념 , 문법 , computational thinking

  2. 코딩 스킬

    숙달의 영역 : 의도적 연습 , API , Framework / Library

  3. 프로젝트

    경험의 영역 : 미니 프로젝트를 통해 가상 경험
