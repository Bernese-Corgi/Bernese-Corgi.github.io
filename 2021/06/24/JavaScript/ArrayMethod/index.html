배열 메서드

배열 메서드 개요

Array / Array.prototype

자바스크립트는 다양한 배열 빌트인 메서드를 제공한다.

  1. Array 생성자 함수
    • 정적 프로퍼티 : Array.length
    • 정적 메서드를 제공한다 : Array.from(), Array.isArray(), Array.of()
  2. Array.prototype
    • 모든 인스턴스는 Array.prototype의 메서드를 상속받아 사용할 수 있다.
    • 프로토타입 프로퍼티 : Array.prototype.constructor, Array.prototype.length
    • 프로토타입 메서드를 제공한다.

mutator / accessor

  1. mutator

    • 원본 배열을 직접 변경하는 메서드
    • 외부 상태를 직접 변경하는 부수효과가 있다.

  2. accessor
    • 원본 배열을 직접 변경하지 않고 새로운 배열을 생성하여 반환하는 메서드

❓ 원본 배열

  • 배열 메서드를 호출한 배열.
  • 배열 메서드의 구현체 내부에서 this가 가리키는 객체

Array 정적 메서드

Array.of

  • 전달된 인수를 요소로 가지는 배열을 생성한다.
  • Array.of는 Array 생성자 함수의 정적 메서드이다.
  • 인수가 1개이고 숫자더라도 인수는 요소로 동작한다.

Array.from

  • 유사 배열 객체 or 이터러블 객체를 인수로 전달받아 배열로 변환하여 반환한다.
  • Array.from은 Array 생성자 함수의 정적 메서드이다.

  1. 유사 배열 객체를 배열로 변환하여 반환한다.

    length만 존재하는 유사 배열 객체를 전달하면 undefined를 요소로 채운다.

    | 유사 배열 객체 |

    • 배열처럼 인덱스로 프로퍼티 값에 접근할 수 있고, length 프로퍼티를 갖는 객체, 순회할 수 있다.

    • 유사배열 객체는 for문으로 순회할 수 있다.

  2. 이터러블을 배열로 변환하여 반환한다.
    문자열은 이터러블이다.

    | 이터러블 객체 |

    • Symbol.iterator 메서드를 구현하여 for…of 문으로 순회할 수 있으며, 스프레드 문법과 배열 디스트럭처링 할당의 대상으로 사용할 수 있는 객체

    • Array, String, Map, Set, DOM 컬렉션(NodeList, HTMLCollection), arguments

  3. 두 번째 인수로 콜백함수를 전달할 수 있다
    값을 만들면서 요소를 채울 수 있다.

Array.isArray

  • Array 생성자 함수의 정적 메서드

  • 배열인지 아닌지 판단하여 true, false를 반환한다.

  • 인수 : 배열인지 아닌지 판단할 것

  • 반환값 : boolean. 인수가 배열이면 true, 배열이 아니면 false
    ① true

    ② false

Array 프로토타입 메서드

변경자 메서드 (mutator)

변경자 메서드는 원본 배열을 수정한다.

Array.prototype.push

  • 배열의 끝에 하나 이상의 요소를 추가하고, 배열의 새로운 길이를 반환한다.
  • 인수 : (배열의 끝에 추가할 요소[, ... [, 배열의 끝에 추가할 요소]])
  • 반환값 : 변경된 배열의 새로운 length 속성
  • mutator : 원본 배열을 직접 변경한다.

① 인수로 전달받은 모든 값을 원본 배열의 마지막 요소로 추가한다.

② 변경된 배열의 갱신된 length 프로퍼티 값을 반환한다.

③ 원본 배열이 직접 변경되는 부수 효과가 발생한다.

④ 스프레드 문법으로 대체할 수 있다.

Array.prototype.pop

  • 배열에서 마지막 요소를 제거하고 그 요소를 반환한다.
  • 인수 : 없음
  • 반환값 : 배열에서 제거한 요소를 반환한다. 빈 배열이면 undefined를 반환한다.
  • mutator : 원본 배열을 직접 변경한다.

Array.prototype.unshift

  • 인수로 전달받은 모든 값을 원본 배열의 선두에 요소로 추가하고, 변경된 length 프로퍼티 값을 반환한다.
  • 인수 : 배열 맨 앞에 추가할 요소(들)
  • 반환값 : 메서드를 호출한 배열의 새로운 length 속성
  • mutator : 원본 배열을 직접 변경한다.

Array.prototype.shift

  • 배열에서 첫 번째 요소를 제거하고, 제거된 요소를 반환한다.
  • 배열의 길이 length의 변경이 발생한다.
  • 인수 : 없음
  • 반환값
    ① 원본 배열이 빈 배열이 아니면 배열에서 제거된 첫 번째 요소
    ② 원본 배열이 빈 배열이면 (length 값이 0이면) undefined를 반환한다.
  • mutator : 원본 배열을 직접 변경한다.

▾ while 반복문 안에서 shift() 사용하기

Array.prototype.splice

  • 배열의 기존 요소를 삭제 or 교체하거나 새 요소를 추가하여 배열의 내용을 변경한다.
  • 배열의 중간에 요소를 추가하거나 중간에 있는 요소를 제거하는 경우 사용한다.
  • 인수
    ① 변경을 시작할 인덱스
    ② 배열에서 제거할 요소의 개수 (선택)
    ③ 배열에 추가할 요소 (선택)
  • 반환값 : 제거한 요소를 담은 배열을 반환한다.

① 첫 번째 인수 : 변경을 시작할 인덱스

  • 배열의 길이보다 큰 값이라면 실제 시작 인덱스는 배열의 길이로 설정된다.

    ② 배열에서 제거할 요소의 개수 (선택)
    ③ 배열에 추가할 요소 (선택)

  • 반환값 : 제거한 요소를 담은 배열을 반환한다.

접근자 메서드 (accessor)

접근자 메서드는 배열을 수정하지 않고, 기존 배열의 일부에 기반한 새로운 배열 또는 값을 반환한다.

Array.prototype.indexOf

  • 원본 배열에서 요소를 검색하여 인덱스를 반환한다.
    ⇒ 검색한 요소에 해당하는 인덱스 or -1
  • 인수
    ① 배열에서 찾을 요소
    ② 검색을 시작할 인덱스 (선택)
  • 반환값 : 검색한 요소에 해당하는 인덱스 or -1

① 중복되는 요소가 여러 개 있어도 첫번째로 검색된 요소의 인덱스 하나만 반환한다.

② 요소가 존재하지 않으면 -1을 반환한다.

③ 두번째 인수로 검색을 시작할 인덱스를 전달할 수 있다. 생략할 경우 처음부터 검색한다.

④ indexOf는 배열에 특정 요소가 존재하는지 확인할 때 유용하다.

⑤ Array.prototype.includes 메서드로 대체할 수 있다.

Array.prototype.concat()

  • 인수로 전달된 배열이나 값들을 기존 배열에 합쳐서 새로운 배열을 반환한다.
  • 인수
    ① 배열 or 값
    ② 인수를 생략하면 기존 배열의 얕은 복사본을 반환한다.
  • 반환값 : 새로운 배열 객체
  • accessor : 원본 배열이 변경되지 않는다.

① 인수가 배열인 경우 : 배열을 해체하여 새로운 배열의 요소로 추가한다.

② 인수가 특정 값인 경우 : 인수로 전달한 값을 새로운 배열의 마지막 요소로 추가한다.

③ 인수에 배열과 특정 값을 함께 전달하는 경우 : 배열과 특정 값을 마지막 요소로 추가한 새로운 배열을 반환한다.

④ push, unshft 메서드를 대체할 수 있다.
